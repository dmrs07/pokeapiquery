# Pokemon Query Challenge:

## Usage

- Clone the Repo 
```sh
git clone git@github.com:dmrs07/pokeapiquery.git
```
### Then:

- Run _npm install_
- Run _npm start_
- Make a request to http://localhost:3000/pokemon/:id, assuming id will be the pokemon to be queried.
